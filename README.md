# 書籍レビューアプリ

## 開発環境

| ツール  | バージョン |
| ------------- | ------------- |
| node.js  | v16.19.1  |

* 認証トークンを設定し、書籍一覧取得APIを使って、書籍レビューの一覧を表示
* ページネーション実装
